# Social Media

Social media content produced at the Environmental Inequality Lab.

## Folder Convention

Each campaign or project gets its own folder named `YYYY-[topic-slug]` (e.g., `2026-epa-brief-launch`).

```
YYYY-[topic-slug]/
├── assets/        # source images, graphics, or design files
└── copy/          # caption drafts and text content
```

## Formats and Style

Image templates live in `formats/social/`. Caption and platform conventions are in `style-guides/social/`.
