# Social Media

Social media content produced at the Environmental Inequality Lab that is **not tied to
a single paper** — lab news, events, hiring, cross-paper campaigns.

**Social tied to a paper lives with that paper**, not here: `papers/<paper-slug>/social/`,
alongside the paper's `press-release/`, `data-viz/`, and `blog/` (see `papers/README.md`).
This folder is only for social that has no home under one paper.

## Folder Convention

Each non-paper campaign gets its own folder named `YYYY-[topic-slug]` (e.g., `2026-annual-report`).

```
YYYY-[topic-slug]/
├── assets/        # source images, graphics, or design files
└── copy/          # caption drafts and text content
```

## Formats and Style

Image templates live in `formats/social/`. Caption and platform conventions are in `style-guides/social/`.
