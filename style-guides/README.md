# Style Guides

Writing and design conventions for EIL outputs — organized by output type.

## Subfolders

| Folder | Purpose |
|---|---|
| `research-highlight/` | Tone, citations, figures, and formatting rules for research highlights |
| `press-release/` | Writing conventions for press releases |
| `blog/` | Tone and formatting conventions for blog posts |
| `social/` | Caption style, image guidelines, and platform conventions |
| `newsletter/` | Tone and formatting conventions for newsletter posts |
| `data-viz/` | Design and annotation conventions for standalone visualizations |
