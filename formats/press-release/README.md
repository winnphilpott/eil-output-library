# Press Release Templates

Starter templates for EIL press releases. Copy a template into a new paper's `press-release/` folder and fill in the placeholders.

## Available Templates

| File | Description |
|---|---|
| `press-release-template.qmd` | One-page press release with release line, media contact, headline, dateline + lead, body, quotes, end marker, and boilerplate |

## How to Use

1. Copy the template into your new paper's `press-release/` folder and rename it to `[topic-slug]_press-release.qmd`
2. Update `include-in-header` in the YAML to point to `../../../formats/research-highlight/_style.tex` (the shared design system)
3. Update the logo path to `../../../formats/logos/eil-logo-maroon.png` (or white version)
4. Fill in all `[placeholder]` text with your content
5. Confirm every attributed quote with the speaker before publishing

See `style-guides/press-release/README.md` for the full writing conventions.
